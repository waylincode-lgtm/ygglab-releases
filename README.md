# YGG Lab

> 一个桌面。你的 AI Agent 们,真的在协作。

YGG Lab 是一款桌面应用 —— 让多个 AI Agent 住在你自己的电脑上,单独干活、组队协作,或被你远程指挥,7×24 小时不停地把事情往前推。对话与任务记录只在本机,从不离开你的电脑。

官网:<https://ygglab.ygg.com.cn>

## ✨ 核心能力

- **Agent 协作** —— 20+ 款 CLI Agent 与 30+ 模型平台协同工作
- **内置助手** —— 20+ 位专业助手(PPT 制作、数据分析、代码编程……)开箱即用
- **远程控制** —— 微信、飞书、钉钉、Telegram、WebUI 多渠道远程派活
- **定时任务** —— 7×24 周期性调度,任务完成自动把结果推到你的 IM 频道
- **办公文件** —— 原生生成 PPT、Excel、Word、PDF
- **技能市场** —— 可扩展 Skills 生态 + 统一 MCP 配置
- **隐私优先** —— 100% 本地运行,你的数据从不上云

## 🚀 三步开始

1. **一分钟装好** —— 支持 Windows、macOS,开箱即用
2. **自动发现 Agent** —— 已装好的 Claude Code、Gemini CLI 等直接接入
3. **像聊天一样派活** —— 产物直接落到电脑,随时二次编辑

## 📥 下载

前往 **[Releases](../../releases)** 下载最新版本:

| 平台 | 文件 |
|------|------|
| Apple Silicon Mac(M 芯片) | `YggLab-*-mac-arm64.dmg` |
| Intel Mac(2020 年前) | `YggLab-*-mac-x64.dmg` |
| Windows x64 | `YggLab-*-win-x64.exe` |

- **macOS**:打开 dmg → 拖「YGG Lab」进「应用程序」(已签名+公证,双击直开)
  - 区分芯片: → 关于本机 → Apple M… 用 arm64;Intel 用 x64
- **Windows**:下载 exe 直接运行安装
